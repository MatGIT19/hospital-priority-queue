# data-structures-priority-queue-doubly-linked-list
## Key Features
Doubly linked list implementation (from scratch)

Dynamic insertion based on priority rules

Single unified queue with logical priority partition

Efficient removal (O(1)) from the head

Search functionality by patient name

Interactive CLI menu for testing
